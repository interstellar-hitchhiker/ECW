# Constant Point in an Ever-Changing World

A small browser animation about constancy, motion, and perspective. A fixed red **X** remains the reference point while particles, trails, and ripples move around it, showing how change becomes readable only when measured against something stable.

## Version 2

Version 1 was a simple 2D p5.js sketch: an **X** stayed in the centre while bouncing particles moved around it.

Version 2 rebuilds the idea as a stronger interactive Canvas animation. It adds visible motion trails, ripple disturbances, adjustable energy, time speed, particle density, and different viewing frames, so the fixed point becomes an active reference system rather than just a static symbol.

## Why it exists

The project is a visual metaphor: in a changing world, a constant point helps reveal what is moving, what is shifting, and what remains stable.
